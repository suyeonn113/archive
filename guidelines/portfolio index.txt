Typography Scale

H1 (페이지 타이틀)
글자 크기: 32–36px
줄간격: 120–130%
사용 용도: 섹션 제목, 페이지 헤더

H2 (섹션 제목)
글자 크기: 24–28px
줄간격: 130%
사용 용도: 기능 및 챕터 구분

H3 (소제목)
글자 크기: 20px
줄간격: 140%
사용 용도: 화면별 설명 제목

본문 (Body)
글자 크기: 16px
줄간격: 150%
사용 용도: 기획 의도, 설명 문장

보조 설명 (Secondary Text)
글자 크기: 14px
줄간격: 150–160%
사용 용도: 추가 설명, 부연 텍스트

캡션 (Caption)
글자 크기: 12–13px
줄간격: 140–150%
사용 용도: 이미지 설명

라벨 / UI 텍스트 (Label / UI Text)
글자 크기: 13–14px
줄간격: 140%
사용 용도: 버튼, 태그

메타 정보 (Meta Information)
글자 크기: 12px
줄간격: 140%
사용 용도: 날짜, 상태값
